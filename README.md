# 🌌 AuraMatch — Quantum Affinity Resonance Engine

AuraMatch ek modern, high-end Web Application hai jo cryptographic-style deterministic mathematical algorithms ko use karte huway do names ke darmiyan affinity aura (love percentage) calculate karti hai. Isme advanced glassmorphism UI, pseudo-3D parallax interaction cards, aur real-time cloud data logging ka istamal kiya gaya hai.

## ✨ Features (Khasusiyat)

* **🔮 Deterministic Resonance Engine:** Algorithm is tarah design kiya gaya hai ke yeh universally bi-directional hai. Agar aap `A + B` check karein ya `B + A`, result hamesha 100% same aur accurate aayega.
* **📡 Real-Time Cloud Logging:** Google Firebase Firestore ke sath integrated hai. Jab bhi koi user names enter karta hai, data background mein securely store ho jata hai.
* **🎨 Dynamic Theme Engine:** System automatic ya manual user preference ke mutabiq Dark aur Light mode mein switch ho jata hai.
* **✨ Pseudo-3D Parallax Effects:** Advanced CSS variables aur JavaScript mouse tracking ka use karte huway card par ek immersive 3D glass effect dikhai deta hai.
* **📱 Ultra-Responsive UX:** Mobile, tablet aur desktop screens ke liye mukammal optimized interface aur smooth animation counters.

---

## 🛠️ Tech Stack (Inpage Technologies)

* **Frontend:** Semantic HTML5, Core CSS3 (Custom Variables, Media Queries)
* **Logic Engine:** Vanilla JavaScript (ES6+, RequestAnimationFrame API, Cubic Easing Curves)
* **Backend/Database:** Google Firebase (Firestore Database SDK - Compat Layer)
